# 🧑‍💻 Matías Lucero — Portfolio Personal

Sitio web personal donde presento mi perfil como **Desarrollador FullStack**, mis proyectos, experiencia y stack tecnológico.

🔗 **[Ver demo en vivo](https://matt8421.github.io/PortfolioMatiasLucero/)**

---

## ✨ Highlights

- Cursor personalizado con **partículas de destello** interactivas
- Fondo animado con **red de nodos**, pulsos expansivos y **lluvia de meteoros**
- Animaciones de entrada por scroll con `IntersectionObserver`
- Efecto **split-text** letra a letra en el hero
- Diseño **100% responsive** con navegación mobile
- Formulario de contacto funcional via **Formspree**
